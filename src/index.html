simplicity

A collection of simple css examples

Simple Page Demo

A basic, one-column, centred page. This index page uses a variation of this style.

Fixed Header Demo

A one-column page with a fixed header which extends across the entire viewing area.

Fixed Narrow Header Demo

A one-column page with a classic narrow fixed header which extends across the entire viewing area.

Sidebar Navigation Demo

A two-column page where the left-hand column is used for site navigation. For browsers with a small viewing area (such as mobile browser) the navigation area is moved above the main section instead, and the navigation links are re-aligned horizontally.

Three-Section Page Demo

A three-section page which is similar in layout to the Sidebar Navigation page above, but which fills the entire viewing area.

Multi-column Layout Demo

A basic multi-column layout, in this example using three columns.

Three-Section, Fixed Header Page Demo

A three-section page which is similar in layout to the Sidebar Navigation page above, but which fills the entire viewing area, and where both the header and the navigation sidebar are fixed, so that only the content area scrolls.

Magazine Page Demo

A basic two-column magazine-style layout, using the new 'column' CSS feature (so this might not work on all browsers).
